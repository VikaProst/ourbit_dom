# SQUAD TERMINAL — Chrome-расширение (авто-подключение Ourbit)

Считывает токен `uc_token` с открытой страницы **Ourbit** и автоматически подключает
локальный терминал (`http://localhost:8777`). Друзьям не нужно вставлять токен руками.

Токен никуда наружу не уходит — только на **твой localhost:8777**.

## Установка (как у GC-TERMINAL)

1. Запусти локальный терминал (server.py, порт 8777).
2. Chrome → **⋮ → Расширения → Управление расширениями**.
3. Включи справа вверху **«Режим разработчика»**.
4. Нажми **«Загрузить распакованное»** → выбери папку `chrome-extension`.
5. В панели Chrome появится иконка расширения.

## Использование

1. Войди в аккаунт на **futures.ourbit.com** и оставь её вкладку открытой.
2. Открой терминал `http://localhost:8777`.
3. Нажми иконку расширения → **«Подключить терминал»**.
4. Расширение прочитает токен и подключит терминал (увидишь баланс + fee).

## Несколько аккаунтов / ADS-профили

Каждый профиль браузера / ADS = отдельная установка расширения. В каждом профиле
оно берёт токен того аккаунта, который залогинен в этом профиле.

## Если токен не находится

Расширение ищет cookie/localStorage с именами вроде `uc_token`, `token`, `authorization`.
Если у Ourbit другое имя — напиши, какое (F12 → Application → Cookies/LocalStorage на
странице Ourbit), и я добавлю его в список (`COOKIE_NAMES`/`LS_HINT` в `popup.js`).

## Файлы
- `manifest.json` — манифест MV3 (permissions: cookies/scripting/tabs, host: ourbit.com + localhost:8777).
- `popup.html` / `popup.js` — кнопка «Подключить» + чтение токена + POST на `/api/exttoken`.

## Сервер
Терминал принимает токен на `POST /api/exttoken` (разрешён origin `chrome-extension://`),
дальше — обычный `_trade_connect` (баланс + проверка 0% fee).
