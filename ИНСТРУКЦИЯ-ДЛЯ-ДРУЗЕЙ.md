# 🐸 SQUAD TERMINAL — установка (для друзей)

Терминал работает **у тебя на компьютере** (localhost). Твой токен и данные **никуда не уходят** — только на твой ПК и на биржу Ourbit напрямую. Никто, кроме тебя, их не видит.

Нужно сделать 3 вещи: **① поставить терминал ② поставить расширение в Chrome ③ подключиться.**

---

## ① Терминал (5 минут, один раз)

1. **Установи Python** (если ещё нет):
   - Скачай с https://www.python.org/downloads/ → «Download Python 3.x».
   - Запусти установщик. **ОБЯЗАТЕЛЬНО поставь галочку внизу «Add Python to PATH»**, потом «Install Now».

2. **Распакуй папку** `ourbit_dom`, которую тебе прислали, в удобное место (например на Рабочий стол).

3. **Запусти терминал**: зайди в папку `ourbit_dom` и **двойной клик по `start.bat`**.
   - Откроется чёрное окно — оно ставит библиотеки (первый раз ~минуту) и запускает терминал.
   - Сам откроется браузер на `http://localhost:8777` — это и есть терминал.
   - ⚠️ **Чёрное окно не закрывай, пока торгуешь.** Закрыл окно = выключил терминал.

> Если Windows ругается «Защита компьютера» → «Подробнее» → «Выполнить в любом случае» (файл локальный, безопасный).

---

## ② Расширение в Chrome (2 минуты, один раз)

Расширение само вытаскивает твой токен с открытой страницы Ourbit и подключает терминал — не надо ничего копировать руками.

1. Открой Chrome → в адресной строке набери: **`chrome://extensions`** → Enter.
2. Вверху справа включи тумблер **«Режим разработчика» / «Developer mode»**.
3. Нажми **«Загрузить распакованное расширение» / «Load unpacked»**.
4. Укажи папку **`ourbit_dom/chrome-extension`** (внутри присланной папки).
5. Появится иконка **🐸 SQUAD TERMINAL** рядом с адресной строкой (может прятаться под значком пазла 🧩 — закрепи её кнопкой-булавкой).

---

## ②.5 🔐 Вход по логину и паролю (обязательно)

Терминал закрыт логином и паролем — их тебе даёт Вика (каждому свои).

1. Когда откроется `http://localhost:8777`, ты увидишь **экран входа** 🐸 SQUAD TERMINAL.
2. Впиши **логин** и **пароль**, которые прислала Вика → нажми **«Войти»**.
3. Всё — терминал откроется. Вводить нужно при каждом запуске `start.bat`.

> ⚠️ Логин/пароль — только твои, не передавай их другим. Если Вика их отзовёт — вход перестанет работать.
> Файл `license.txt` больше не нужен (старая система ключей).

---

## ③ Подключиться и торговать

1. Зайди на **https://futures.ourbit.com**, **войди в свой аккаунт** (важно — залогинься).
2. Убедись, что **терминал запущен** (чёрное окно из шага ①, вкладка `localhost:8777` открыта, вход выполнен).
3. Нажми иконку расширения **🐸** → кнопка **«Подключить»**.
   - Должно написать `✓ подключено · баланс $…`.
4. Иди во вкладку терминала `localhost:8777` → нажми галочку **LIVE** (станет зелёной) → можно торговать.

## 🎮 Управление (как в MetaScalp)

| Действие | Как |
|---|---|
| Купить по рынку | клавиша **T** / ЛКМ по красной (аск) |
| Продать по рынку | клавиша **Y** / ПКМ по зелёной (бид) |
| Лимит-покупка | ЛКМ по своей стороне (бид) |
| Лимит-продажа | ПКМ по своей стороне (аск) |
| **Закрыть позицию** | клавиша **D** или **Alt+клик** |
| **Заявка на закрытие (лимит)** | **зажми F + клик** по цене (или Ctrl+клик) |
| Отменить одну лимитку | клик по красному **×** или **колёсиком** по ней |
| Отменить все лимитки | **Пробел** или **B** |
| Реверс | клавиша **R** |
| Синхронизация с биржей (↻) | кнопка ↻ — покажет все стоящие заявки/позицию |

## ❓ Если что-то не так

- **«неверный логин или пароль»** → проверь, что вписала ровно то, что прислала Вика (без лишних пробелов). Если не помогает — попроси Вику проверить/выдать заново.
- **«сервер входа недоступен»** → проверялка входа временно недоступна, напиши Вике. Интернет у тебя при этом должен быть включён.
- **«терминал недоступен (localhost:8777)»** → не запущен терминал. Открой `start.bat` (шаг ①).
- **«токен не найден»** → зайди на futures.ourbit.com и **залогинься**, оставь вкладку открытой, жми «Подключить» снова.
- **баланс $0 / токен истёк** → перелогинься на Ourbit, нажми «Подключить» ещё раз (токен Ourbit протухает).
- **расширение пропало после перезапуска Chrome** → так и должно быть в режиме разработчика; просто снова нажми на иконку и «Подключить» (переставлять не надо).
